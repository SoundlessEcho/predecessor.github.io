Getting Started

Welcome to Predecessor. This toolkit includes a loader, HUD FX, and utilities tailored to Call of Duty GSC modding.

git clone https://github.com/YourOrg/Predecessor.git
cd Predecessor

Installation

  1. Download the latest release from the Downloads page.
  2. Extract to a clean folder.
  3. Follow the README for the target COD title.

API / Functions

Example HUD text creation:

CreateText(player, text, x, y, scale, color)

Decoder-style animation (pseudo):

for (i = 0; i < text.length; i++) { /* animate */ }
